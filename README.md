Thiranex Task 2: Predictive Modeling (Machine Learning)

This project focuses on building a supervised machine learning model to predict outcomes based on historical data using the Titanic dataset.

🎯 Project Goals

Apply machine learning algorithms (Random Forest).

Perform feature engineering (Encoding categorical variables).

Evaluate model performance using professional metrics.

🤖 Model Performance

Algorithm: Random Forest Classifier

Accuracy: 81.56%

Evaluation: Detailed performance visualized through a Confusion Matrix.

🛠️ Tech Stack

Scikit-Learn: For model training and evaluation.

Pandas: For feature engineering.

Seaborn: For performance visualization.

📊 Visualizations

Confusion Matrix: Shows the ratio of true positives vs. false negatives.

Feature Importance: Identifies that Sex, Fare, and Age were the most critical predictors of survival.
